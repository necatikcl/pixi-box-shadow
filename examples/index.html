Benchmark 01

Animate shadow color in-place

The box geometry stays fixed while the shadow color is updated every frame by writing directly to the filter uniforms.

–

Color sweeps from red to blue and back again. This isolates the cost of mutating shadow properties without forcing geometry recalculation.

Benchmark 02

Animate geometry and shadow bounds

The rectangle expands and contracts continuously, exercising the path where the element dimensions and shadow coverage must be re-evaluated.

–

Width and height animate together. This measures the runtime impact of shape changes that alter shadow placement and padding.
